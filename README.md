# PylaMydd

# THIS IS A FORK OF PYLA AI

PylaMydd is an advanced, optimized, and fully standalone external automation script.

## How to run:
1. Install Python (tested with Python 3.11.9)
2. In a command prompt within the folder, run `pip install -r requirements.txt` to install the necessary libraries.
3. Once completed, run `pip install "adbutils~=2.12.0"`
4. Run start.bat
