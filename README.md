# PylaMydd

PylaMydd is an advanced, optimized, and fully standalone external automation script.

## How to run:
1. Install Python (tested with Python 3.11.9)
2. In a command prompt within the folder, run `pip install -r requirements.txt` to install the necessary libraries.
3. Once completed, run `pip install "adbutils~=2.12.0"`
4. Run `python main.py`
5. Enjoy your fully clean, custom Control Center UI!

## Features
- Hardware Acceleration support (DirectML / CUDA) for lag-free processing.
- Clean and premium user interface (PylaMydd Control Center).
- Stripped from all intermediate overheads to run as smoothly as possible.
